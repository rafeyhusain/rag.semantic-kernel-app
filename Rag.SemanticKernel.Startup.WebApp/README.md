# Rag.SemanticKernel.Startup.WebApp

This project provides a web application entry point for the Semantic Kernel application. It is used for running Semantic Kernel features as a web service.

## Features

- Web-based interface for Semantic Kernel
- Enables HTTP access to Semantic Kernel features

## Usage

Run this project to interact with Semantic Kernel features via a web interface.

## Dependencies

- Microsoft.AspNetCore
- Microsoft.SemanticKernel
